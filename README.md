# DECORA ATELIER

**Premium Home Decor & Interior Styling**

## 📖 Overview

DECORA ATELIER is a modern, fully responsive e-commerce web application for premium home decor products. Built with pure HTML, CSS, and JavaScript, it delivers a seamless shopping experience with cart management, user authentication, and order placement — all powered by localStorage.

**Live Demo:**  https://webapp-eight-sand-89.vercel.app/

---

## ✨ Features

### 🛍️ Shopping
| Feature | Description |
|---------|-------------|
| Product Categories | Wall Art, Plants, Lamps, Vases, Clocks |
| Smart Search | Dropdown category search with icons |
| Product Grid | Responsive cards with Add/Buy buttons |
| Shopping Cart | Quantity controls, dynamic pricing, item removal |

### 👤 Account System
| Feature | Description |
|---------|-------------|
| Sign Up / Login | Full registration with address collection |
| Dashboard | Profile view, order stats, preferences |
| Edit Profile | Update personal info and delivery address |
| Guest Mode | Shop without creating an account |

### 💳 Checkout
| Feature | Description |
|---------|-------------|
| Order Summary | Subtotal + shipping (free over ₹2000) |
| COD Payment | Cash on Delivery only |
| Success Modal | Order confirmation popup |

### 🎨 Design
- Hero slider with auto-rotation
- Smooth animations and hover effects
- Fully responsive (mobile → desktop)
- Premium typography (Playfair Display + Plus Jakarta Sans)
